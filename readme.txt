安装环境需求：PHP >= 5.0; MYSQL >= 5.0

全新安装指南：

1. 上传所有程序，将 database 目录下所有文件设为可写，将 _cache , m/_cache , _static/picture , _static/avatar 目录设为可写！
2. 运行 install.php 安装！
3. 安装成功后别忘了删除 install.php 文件！
4. 进入系统管理进行相关设置。
